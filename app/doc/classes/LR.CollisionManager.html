Defined in app\shared\js\dev\phaser\collision_manager.js:127

Called before allowing collision between two bodies. By returning true, you allow a collision to happen. This is called for a pair ie if onPostBroadphase( player , enemy1) is called, onPostBroadphase( enemy1, player) won't be called

If you want to be able to process this phase, use setPostBroadPhaseCallback(callback,context) on your GameObject
